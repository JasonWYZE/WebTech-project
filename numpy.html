Numpy

numpy is a numerical calculation extension of Python, specially used to deal with matrices, its operation efficiency is more efficient than lists.

Import Numpy

        
          import numpy as np;

Import numy and named it as np, which is much easier to call it later when we are programing. The data structure in numpy is a n-D array. 

        
          data = [[1,2,3],[4,5,6]]
          array = np.array(data2)
          array
          *output
          array([1,2,3],[4,5,6])

As we can see, it will generate a matrix. One point that we need to consider is that all element in an array should be same type(int, float, string). We can use dtype to check what type of data every time. 

        
          array.dtype
          *dtype('int64')

Numpy can do math every easier. For instance

        
          array
          *array([1,2,3,4,5,6])
          array * array2
          *array([1,4,9,16,25,36])

          array2
          *array2([1,2,3],[4,5,6])
          array2+1
          *array([2,3,4],[5,6,7])
          array2[0]
          *array2([1,2.3])
          array2[0][1]
          *2

As we can see array can do multiple calculation and it will change based on the matrix in the array list. Besides those operations, it also include reshape, ufunc, sort and many functions. Here is a video I recommended for beginers to explore more about Numpy.

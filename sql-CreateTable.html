What is a database? What is MySQL?

The database, composed of multiple tables, is used to store data. It is like Excel that suitable for storing some simple text information

  • ID
  • Last Name
  • First Name
  • Phone number
  • Address

However, it is difficult to store some data like

  • Photo
  • Fiction
  • Movie

Those data are too large to store in the database. For instance, for amazon, the information of product, store, and sale will store at database while the image or video for promotion will not store in database.

There are many types of database, relational, NoSQL, Wide column, and Key-value. The most common used one, which is I will introduce later, is MySQl, a relational databases software.

How to Create Database

The first step in studying MySQL is creating a simple table. We can use CREATE DATABASE for creating a database in MySQL 

            
                CREATE  TABLE IF NOT EXISTS `Member` (
                    `member_ID` INT  AUTOINCREMENT ,
                    `full_names` VARCHAR(150) NOT NULL ,
                    `gender` VARCHAR(6) ,
                    `date_of_birth` DATE ,
                    `address` VARCHAR(255) ,
                    `contact_number` VARCHAR(75) ,
                    `email` VARCHAR(255) ,
                    PRIMARY KEY (`member_ID`) )
                    ENGINE = InnoDB;

The code above show how to create a table with multiple columns. It also set up the data type in each column. The primary key in this table is student_ID.

SQL SELECT statement

It is the most frequently used SQL command and it can access the database and select specific information you want.

Some common applications of SELECT:

Looking for all male members in this table

            
              select * from 'student' where 'gender' = 'male'

Output:

As we can see, all the members in this table are male since we add the conditional statement "where". The * refer to all columns. We can also change it to any specific column we are look for. from 'student' refer to which table we are looking since a database was composed of multiple tables.

If we want to see the male member and sort at order by birth date, we can use the code like this: 

        
        select * from 'member' where 'gender' = 'male' 
          order by 'date_of_birth' ASC

There are many way to use MySQL. For instance, adding limit at the end will let the query show limit amount of result. Group by is another way to show the data. For instance, if you are looking for different gender of member, you can use 'group by 'gender''. Another useful way in MySQL is join, which will combine multiple tables together so you can access data from different tables.

What is this website about

This website is about my personal learning experience in data analyst. It will introduce what skills you need to learn in order to be a data analyst. I will introduce the three main packages in python for data analyst, which are numpy, pandas, and Matplotlib. Besides, it also include the reflection and code of my previous data analyst project.

I build this website for sharing my personal experience since I was very confused when I frst start self learn those skills. By sharing those experiences and teach to visitors, it not only help me further understand the model I used in my project, but also help others who is interested in data analyst.

Hope you guys enjoy!

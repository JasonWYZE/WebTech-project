Pandas

Pandas is the one of most useful package in the python that used for data analyst. By reading data file through index and columns, it allow us to process and deal with data. We can quickly apply complex conversion or filter data in this package. It is the first Python package I learned when I started my data analyst adventure.

Pandas, Numpy, and Matplotlib are three most important and base package in data analyst. Sklearn is another a strong package for machine learning and build models. However, I believe everyone who is interested in data analyst can learn a lot in these three packages.

Import Pandas

        
          import pandas as pd;

This is the standard way that importing pandas and naming it as pd since it is simpler. However, before importing it, make sure you already download and install with Anaconda.

How to import data in pandas and data Type

        
          df = pd.read_csv('titanic.csv');

The code above show the way how to read a csv file. By reading this csv file and stored it in the dataframe and named it df. I used the famous titanic case that predict the factor influence who survivied at the end. This csv file you can find in Kaggle,

There are two types of data in Pandas, series and dataframe. Series is a 1-D data type and each of element is with a label. Dataframe is a 2-D data structure. It can stored multiple type of data.

We can start to check this data file before we analyze it. For instance, if we want to look at the first x rows data or last x rows. 

        
          df.head(10);
          # read the first 10 rows
          df.tail(10);
          # read the last 10 rows

There are multiple way to access a single column in a dataframe.

        
          df.gender;
          df['gender'];

Both code above can access a single series in gender column.

Data Cleaning

Usually, when we are dealing with dataset, there are usually some missing value

        
          df.dropna();

The code above will delete the rows with any missing value. However, if the data size is too samll, we should not do that. The another way is imputing miss data with mean or median.
